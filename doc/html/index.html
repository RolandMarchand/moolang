About Moolang

Moolang is an educational programming language designed and written by Moowool L. Summerwheat to teach developers C programming and compiler design.

Moolang aims for code quality first and foremost, and makes use of excellent basic and advanced practices to optimize the educational value of the codebase.

Compiling

Moolang uses CMake and is written for Linux with glibc. TODO: find the minimum compatible build.

To build, run those commands in order.

cd /path/to/moolang
mkdir build
cmake -S . -B build
cd build
make -j$(nproc)
